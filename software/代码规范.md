# 函数

-   函数命名使用驼峰法。

例如：`SearchLine();`

-   函数声明结束后，应该在后面注释该函数作用
-   函数定义时，应该在该函数上面写一段注释块

```c
/***********************************************
* @brief : 描述
* @param : 参数
* @return: 返回值
* @date  : 修改日期
* @author: 作者
************************************************/
```



# 变量

-   变量命名使用小写字母加下划线组成。

例如: `left_line[4]`

# 宏定义

-   宏定义命名使用大写字母加下划线组成

例如：`#define MAX_ROW 5`
